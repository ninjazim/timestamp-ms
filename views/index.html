Timestamp Microservice.

Pass a date string as a parameter to this url. Receive the unix and natural language date.

Example

https://spiffy-trowel.glitch.me/1509685200000 || https://spiffy-trowel.glitch.me/November%203,%202017

Returns

{"unix":1509685200000,"natural":"November 3, 2017"}
